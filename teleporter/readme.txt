:: Teleporter Script ::
Key:                   F5
Up/Down Arrows:        Navigation
Enter: 	               Confirm Option
Delete Key, Numpad 0:  Return back to previous menu
ESC: 	               Close the menu 

:: REQUIREMENTS ::
• Watch_Dogs 2 Script Hook (Latest Version)

:: INSTALLATION ::
• Place the teleporter folder in: WATCH_DOGS2\bin\ScriptHook\data\scripts

:: RECOMMENDED MODS ::
• Boundary Removal - https://www.nexusmods.com/watchdogs2/mods/8
